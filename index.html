Welcome to the Java Lime page!

Java Lime (also known as JLime) is a brand new scripting language developed by evil masterminds Jared (@Coolway99) and Paul (@hotrodman106). Java Lime is based on scripting languages such as LUA and BASIC but uses a IRC inspired syntax. There are currently two versions available for download. The first version is the Android version. This version is designed to run on Ice Cream Sandwich to Lollipop and also on lower ram devices. The second version is designed to be run on Windows computers. While a Mac and Linux version are in the works, they are currently unavailable for the public.

Current Features

  • Beautiful and easy to use GUI
  • Primitive data storage
  • Time and date commands
  • 'If' statements
  • 'For' statements
  • Basic loop command
  • A variety of debug commands

Planned Features

  • Support with microcontrollers such as Arduinos or Raspberry Pi's
  • Basic GUI support
  • Modules
  • User inputs

Authors and Contributors

Java Lime is created by the evil masterminds Paul (@hotrodman106) and Jared (@Coolway99). In the begnning, Java Lime was developed as a way to parse commands for frequently used lines of Java code. However, as time went on, Paul and now fellow developer Jared realized that Java Lime had the potential to become a powerful language and soon it was further devolved creating the language we now love today.
